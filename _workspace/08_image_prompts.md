## 레이아웃 빌더 전달 사항
- 이미지는 초기 단계에서는 CSS 플레이스홀더로 대체
- 구역 배경은 CSS gradient + 이모지로 임시 구현 가능
- 캐릭터/별이는 이모지 또는 SVG 아이콘으로 초기 구현
- 실제 픽셀아트 에셋은 이미지 생성 후 교체
