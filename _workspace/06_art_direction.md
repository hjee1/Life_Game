# 아트 디렉션 — 스토리 블루프린트

## 비주얼 개요
- **제목**: Life_Game — 스테이지 시티
- **아트 스타일**: 소프트 픽셀아트 (16~32px 기반, 부드러운 라인)
- **톤**: 따뜻하고 코지. 노스탤직하되 밝다. "이미 괜찮은 사람의 여정"
- **핵심 감성**: Stardew Valley의 포근함 + 셀레스트의 성취감 + 나이트 인 더 우즈의 감성
- **타깃 감정**: 플레이할 때마다 "나 잘하고 있구나" 느낌

## 캐릭터 디자인

### 메인 캐릭터 (지현우 아바타)
- **체형**: 키 큰 마른 체형 (픽셀아트에서 다른 NPC보다 약간 키가 큼)
- **헤어**: 짧은 검은 머리, 약간 구불거림
- **기본 의상**: 편한 후드+청바지 (일상) / 운동복 (단련의 숲) / 셔츠 (코드 공방) / 무대 의상 (극장)
- **표정**: 기본 미소. 쿨하지만 따뜻한 눈
- **애니메이션**: 걷기, 달리기, 운동, 코딩, 연기, 프리와 산책
- **특별**: 스포트라이트 게이지에 따라 캐릭터 주변 빛 발광 (낮음=약한 빛, 높음=골드 아우라)

### 별이 (AI 컴패니언)
- **형태**: 작은 별 (5각별, 약 캐릭터 머리 크기의 1/3)
- **색상**: 기본 새벽 골드. 감정에 따라 변화:
  - 평온: 노란빛
  - 기쁨: 주황빛 + 반짝임
  - 걱정: 파란빛
  - 감동: 핑크빛
  - 응원: 밝은 흰빛 + 파티클
- **위치**: 항상 플레이어 옆이나 위에 떠다님
- **애니메이션**: 통통 튀기, 회전, 깜빡임, 커지기(감동 시)

### 프리 (푸들)
- **형태**: 작고 하얀 푸들. 픽셀아트에서 귀엽게
- **색상**: 흰색 + 크림, 분홍 혀
- **애니메이션**: 꼬리 흔듦, 달리기, 코 킁킁, 무릎 위 올라감, 아이템 물어옴

### 5인 감독
| 감독 | 외형 키워드 | 컬러 | 상징물 |
|------|-----------|------|--------|
| 인(仁) | 따뜻한 미소의 여성, 숄 | 크림+따뜻한 갈색 | 열린 손바닥 |
| 금(今) | 단단한 체격의 남성, 코트 | 차콜+네이비 | 모래시계 |
| 양(陽) | 밝은 웃음의 남성, 캐주얼 | 주황+노랑 | 해바라기 |
| 문(門) | 지혜로운 노인 여성, 지팡이 | 보라+은색 | 열쇠 |
| 건(健) | 건장한 노인 남성, 간편복 | 숲 그린+갈색 | 나뭇잎 |

### 가면 극단 (과거의 그림자들)
- **디자인 원칙**: 무섭지 않다. 슬프고 약간 투명하다. 과거의 메아리.
- **그림자 소년**: 반투명한 어린 소년, 크고 슬픈 눈, 금 간 가면
- **소음 마스터**: 후드를 쓴 그림자, 주변에 게임 이펙트 잔상
- **비교왕**: 거울을 들고 있는 그림자, 거울에 다른 사람들이 비침
- **공통**: 포용하면 가면이 벗겨지고, 밑에 플레이어와 닮은 얼굴

### UI 원칙
- **미니멀하되 게임적**: 불필요한 정보 없이, 스탯/퀘스트/별이 대사가 핵심
- **원탭 체크**: 데일리 퀘스트 완료는 한 번 탭으로
- **보상 피드백**: 체크 시 파티클 효과 + 스탯 상승 애니메이션 + 별이 반응
- **다크모드**: 밤 시간(21시 이후) 자동 전환, 크림→차콜 배경
- **반응형**: 모바일 우선 (출퇴근 열차에서 사용)

## 구역별 비주얼 컨셉

### 새벽 거리 (Dawn Street)
- **분위기**: 이른 아침 안개, 가로등 불빛, 기차역
- **컬러**: 연보라+골드, 안개 효과
- **특징**: 시간에 따라 안개가 걷히는 애니메이션
- **핵심 오브젝트**: 기차, 가로등, 커피, 책

### 단련의 숲 (Forge Woods)
- **분위기**: 초록 숲, 맑은 시냇물, 바위 훈련장
- **컬러**: 숲 그린+갈색, 햇살 효과
- **특징**: 운동 완료 시 나무가 자라는 애니메이션
- **핵심 오브젝트**: 바위, 나무, 시냇물, 오두막

### 기억의 항구 (Memory Harbor)
- **분위기**: 잔잔한 바다, 정박한 5척의 배, 석양
- **컬러**: 하늘 블루+따뜻한 오렌지, 약간 탈채도
- **특징**: 각 배마다 다른 도시의 실루엣. 방문하면 색이 되살아남
- **핵심 오브젝트**: 배, 등대, 편지, 사진

### 코드 공방 (Code Workshop)
- **분위기**: 따뜻한 공방, 모니터 빛, 도구들
- **컬러**: 코드 퍼플+따뜻한 주황, 모니터 초록빛
- **특징**: 학습 완료 시 공방에 새 도구가 추가됨
- **핵심 오브젝트**: 모니터, 키보드, 톱니바퀴, 책

### 스포트라이트 극장 (Spotlight Theatre)
- **분위기**: 화려한 극장 외관, 따뜻한 연습실, 눈부신 무대
- **컬러**: 극장 레드+새벽 골드, 벨벳 질감
- **특징**: 스탯이 오르면 극장이 점점 화려해짐
- **핵심 오브젝트**: 무대, 조명, 마스크, 커튼

### 별빛 정원 (Starlight Garden)
- **분위기**: 밤하늘 아래 꽃밭, 반딧불, 벤치
- **컬러**: 인연 핑크+새벽 골드, 별빛
- **특징**: 연(緣) 스탯에 따라 꽃이 피어남
- **핵심 오브젝트**: 벤치, 꽃, 다리, 프리, 반딧불

## 레이아웃 빌더 전달 사항
- **모바일 퍼스트**: 375px 기준, 데스크톱 확장
- **핵심 화면**: 대시보드(메인), 구역맵, 퀘스트 상세, 별이 대화, 스탯 상세
- **애니메이션**: CSS transition 기반 (가벼운 파티클은 Canvas)
- **폰트**: Galmuri11 + Pretendard (Google Fonts/CDN)
- **이미지**: 픽셀아트 스프라이트 (향후 생성), 초기는 CSS+이모지 플레이스홀더

## 이미지 프롬프터 전달 사항
- **우선 생성 목록**:
  1. 메인 캐릭터 아바타 (4방향 + 표정 변형)
  2. 별이 (5가지 감정 색상)
  3. 프리 (기본 + 산책 + 아이템)
  4. 5개 구역 배경 (각 1장)
  5. 5인 감독 초상 (각 1장)
- **스타일 키워드**: soft pixel art, cozy, warm lighting, Stardew Valley inspired, Korean modern fantasy
- **금지**: 리얼리스틱, 어두운 톤, 폭력적, 냉소적
