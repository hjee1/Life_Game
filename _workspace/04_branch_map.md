# 분기 구조도

## 분기 개요
- **총 분기점**: 8개 (메인 5 + 히든 3)
- **엔딩 수**: 4개 (메인 3 + 히든 1)
- **분기 구조**: 하이브리드 — 병목 구조(메인) + 상태 누적(스탯) + 폴딩(지연 결과)
- **핵심 원칙**: **실패는 없다** — 어떤 선택도 막다른 길이 아니다. 가면을 다시 써도 벗을 기회가 온다.

## 핵심 분기점

### BP-01: 관객석의 선택 (MQ-01 중)
- **상황**: 별이와 만난 후, 가면 극단의 그림자 소년이 "편하게 살자" 제안
- **선택지**:
  - 🅰️ "무대로 가겠어" — 즉시 스포트라이트 활성화
    - 즉시 결과: 5대 스탯 시스템 해금, 별이 밝아짐
    - 장기 결과: MQ-02 바로 진행 가능
    - 플래그: `STORY_first_choice = STAGE`
  - 🅱️ "좀 더 생각해 볼게" — 잠시 머뭇거림
    - 즉시 결과: 관객석에서 1일 대기, 별이가 기다림
    - 장기 결과: 다음 날 다시 선택 기회 (결국 무대로 감)
    - 플래그: `STORY_first_choice = WAIT`
- **합류 지점**: 어떤 선택이든 MQ-01 완료로 합류 (병목)
- **설계 의도**: 게임이 실패를 허용한다는 것을 첫 순간에 보여줌

### BP-02~06: 기억의 항구 5척 (MQ-02 중)
각 배에서 과거를 직면하거나 회피하는 선택

#### BP-02: 심천호 — 그림자 소년 대면
- **선택지**:
  - 🅰️ "그때는 그랬지. 근데 지금은 달라." → `MASK_shadow_crack += 1`
  - 🅱️ "...아직도 아프다." → `MASK_shadow_empathy += 1`
  - 🅲️ "너도 힘들었겠다." → `MASK_shadow_crack += 2` (최적)
- **합류**: 심천호 퀘스트 완료

#### BP-03: 미시간호 — 외로움 대면
- **선택지**:
  - 🅰️ "그 밤이 있었으니까 지금이 있어" → `INSIGHT_loneliness = ACCEPT`
  - 🅱️ "다시는 혼자 울지 않을 거야" → `INSIGHT_loneliness = RESOLVE`
- **합류**: 미시간호 퀘스트 완료

#### BP-04: 홍콩호 — 소음 마스터 대면
- **선택지**:
  - 🅰️ "아니, 이제 나가야 해" → `MASK_noise_resist += 1`
  - 🅱️ "한 판만..." → `MASK_noise_tempt += 1` (노이즈 지하 진입)
  - 🅲️ "너도 외로웠던 거지?" → `MASK_noise_crack += 2` (최적)
- **합류**: 홍콩호 퀘스트 완료

#### BP-05: 시카고호 — 매튜 편지
- **선택지**:
  - 🅰️ 편지를 쓴다 → `GROWTH_humility = true`, 심(心) +5
  - 🅱️ 편지를 쓰지 않는다 → 나중에 MQ-04에서 다시 기회
- **합류**: 기억의 항구 퀘스트 완료

### BP-06: 첫 오디션 (MQ-03 중)
- **상황**: 준비가 완벽하지 않은 상태에서 오디션에 올라야 하는 순간
- **선택지**:
  - 🅰️ "준비가 안 됐어도 올라간다" → 오디션 진행 (결과는 랜덤하지만 경험치 무조건 획득)
    - 플래그: `COURAGE_audition += 1`
  - 🅱️ "좀 더 연습하고 다음에" → 연습 퀘스트 추가, 다음 오디션 성공률 증가
    - 플래그: `PRACTICE_extra += 1`
- **합류**: MQ-03 첫 공연으로 합류 (어느 경로든 공연은 진행)

### BP-07: 균형의 선택 (MQ-04 중)
- **상황**: 가장 높은 스탯만 올리고 싶은 유혹 vs 약한 스탯 보강
- **선택지**: 동적 생성 — 유저의 실제 스탯 분포에 따라
  - 🅰️ 강한 스탯에 집중 → 스포트라이트 흔들림 지속
  - 🅱️ 약한 스탯 보강 → 균형 달성 → 안정된 빛
- **플래그**: `STAT_balance = true/false` (5대 스탯 편차 기준)

### BP-08: 최종 대면 (MQ-05 중)
- **상황**: 가면 극단 전원 등장. 최후의 유혹
- **선택지**:
  - 🅰️ "싸워서 이기겠어" → 가면 파괴, 강한 엔딩이지만 과거 부정
    - 플래그: `FINAL_approach = FIGHT`
  - 🅱️ "대화로 해결할게" → 가면과 대화, 이해하지만 완전한 포용은 아님
    - 플래그: `FINAL_approach = TALK`
  - 🅲️ "너희도 나의 일부야. 함께 가자" → 가면을 포용, 과거를 인정하며 나아감
    - 플래그: `FINAL_approach = EMBRACE`

## 엔딩 구조

### 엔딩 A: "빛나는 무대" (정석 엔딩)
- **조건**: MQ-05 완료 + `FINAL_approach != EMBRACE` + 5대 스탯 합계 100 이상
- **에필로그**: 무대 위에서 빛나는 자신. 가면은 깨졌지만 과거의 아픔은 남아있다. 그래도 앞으로 나아간다.
- **감정 톤**: 희망적, 약간의 아쉬움

### 엔딩 B: "자기다움의 배우" (최선 엔딩)
- **조건**: MQ-05 완료 + `FINAL_approach = EMBRACE` + 5대 스탯 모두 25 이상 + `STAT_balance = true`
- **에필로그**: 가면을 포용하고, 과거의 자신과 함께 무대에 선다. 열등감도, 방황도, 외로움도 모두 자신의 일부. 완벽하지 않아도 온전한 '나'로서 빛남.
- **감정 톤**: 따뜻한 감동, 카타르시스

### 엔딩 C: "아직 가는 길" (미완의 엔딩)
- **조건**: MQ-05 완료 + 5대 스탯 합계 80 미만
- **에필로그**: 무대에 섰지만 아직 빛이 불안정하다. 하지만 별이가 옆에 있고, 내일도 무대는 열려있다. 여정은 계속된다.
- **감정 톤**: 열린 결말, 희망적

### 히든 엔딩: "별이 된 소년" (진 엔딩)
- **조건**: 엔딩 B + HQ-02 완료 + HQ-03 완료 + 5인 감독 전원 축복 (SQ-09) + 별이 친밀도 100
- **에필로그**: 별이의 정체를 알게 된 후, 자신의 스포트라이트가 처음부터 "사랑"이었음을 깨닫는다. 무대 위에서 별이와 하나가 되어 가장 밝은 빛으로 빛남. 7살의 소원이 진짜로 이루어진 순간.
- **감정 톤**: 눈물나는 감동

```
   MQ-01 완료
      │
      ▼

      │ ┌── 직면 → 가면 crack 누적
      │ └── 회피 → 나중에 재도전 기회
      │     (합류)
      ▼
   MQ-02 완료 ← MASK 플래그 누적
      │
      ▼
   BP-06: 첫 오디션
      │ ┌── 도전 → COURAGE +1
      │ └── 연습 → 다음 성공률 ↑
      │     (합류)
      ▼
   MQ-03 완료
      │
      ▼
   BP-07: 균형 선택 (동적)
      │ ┌── 균형 잡음 → STAT_balance = true
      │ └── 불균형 → 흔들림 지속
      │     (합류)
      ▼
   MQ-04 완료
      │
      ▼
   BP-08: 최종 대면
      │ ┌── FIGHT → 엔딩 A
      │ ├── TALK  → 엔딩 A
      │ └── EMBRACE → 엔딩 B (조건 충족 시)
      │               엔딩 C (스탯 부족 시)
      │
      ▼
   MQ-05 완료
      │
      ├── HQ-02 + HQ-03 + 전조건 → 히든 엔딩
      └── 그 외 → 엔딩 A/B/C
```

## 리플레이 가이드

이 게임은 리플레이가 아니라 **매일의 연속**이다. 엔딩을 본 후에도 데일리 퀘스트는 계속되고, 새로운 사이드 퀘스트가 열린다. "게임은 끝나지 않는다"가 핵심 메시지.

| 상태 | 추천 행동 | 새로운 발견 |
|------|---------|-----------|
| 엔딩 C 도달 | 약한 스탯 보강 후 재도전 | 균형 달성 시 새 대사 |
| 엔딩 A 도달 | 가면 극단 포용 선택지 탐색 | EMBRACE 경로 대사 |
| 엔딩 B 도달 | HQ-02, HQ-03 조건 달성 | 히든 엔딩의 감동 |
| 히든 엔딩 도달 | 게임 계속 — 새로운 퀘스트 해금 | 포스트 엔딩 콘텐츠 |
